# 🎮 Tic Tac Toe

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project allows two players to play the classic Tic Tac Toe game in the browser with a clean and responsive user interface.

---

## 🚀 Features

- 🎯 Two-player gameplay
- 🏆 Automatically detects the winner
- 🤝 Detects a draw when no moves are left
- 🔄 Restart game functionality
- 📱 Responsive design for desktop and mobile
- 🎨 Simple and attractive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Open the project folder

```bash
cd tic-tac-toe
```

3. Open `index.html` in your browser.

No additional installation is required.

---

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns clicking on empty cells.
3. The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click the **Restart** button to play again.
